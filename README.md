# Phoenix

## Phoenix-不死鸟，能自动学习的目录扫描器，碰到重定向的目录会自己记录，并且滚动学习，所以不断的去使用它，可以把它越用越灵活，探测速度不亚于dirsearch

## 新一代目录扫描神器

# 使用说明

首先在将需要字典放在dict.txt文件中，或者在Phoenix目录下将字典改为名为dict.txt

```
-f 导入目标文件，批量扫描

-t 线程

-u 单个目标扫描，接上url 例 http://www.test.com

-c 加入cookie

-d 延迟处理

-mc 匹配状态码

-o 输出扫描日

```
